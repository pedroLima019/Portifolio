# Portfólio Pessoal

Bem-vindo ao meu portfólio pessoal! Este projeto foi criado para apresentar meus trabalhos e projetos de desenvolvimento web. Aqui você encontrará uma coleção dos meus principais projetos, habilidades e uma visão geral do meu progresso na área de tecnologia.

## Tecnologias Utilizadas

Este portfólio foi construído utilizando as seguintes tecnologias:

- **HTML5**: Estruturação das páginas e organização do conteúdo.
- **CSS3**: Estilização e design responsivo para diferentes dispositivos.
- **JavaScript**: Interatividade e manipulação dinâmica de elementos da página.

## Funcionalidades

- **Página Inicial**: Introdução e visão geral dos meus trabalhos.
- **Projetos**: Exibição de projetos com links para demonstrações e repositórios no GitHub.
- **Sobre Mim**: Seção dedicada à minha biografia, experiência e habilidades.
- **Contato**: Formulário de contato funcional para quem deseja entrar em contato comigo.
